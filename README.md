# Stock Price Analysis and Prediction

This is a Streamlit application for analyzing and predicting stock prices. It uses technical indicators and machine learning models to make predictions.

## Features

* Upload CSV file containing stock price data
* Calculate technical indicators (SMA, RSI, MACD, etc.)
* Train a random forest classifier to predict stock price trends
* Make predictions using the trained model
* Display forecast using Prophet

## Requirements

* Streamlit
* Pandas
* NumPy
* TA-Lib
* Prophet
* Scikit-learn
* Matplotlib
* Plotly

## Installation

1. Clone the repository: `git clone https://github.com/your-username/stock-price-analysis.git`
2. Install the requirements: `pip install -r requirements.txt`
3. Run the application: `streamlit run app.py`

## Usage

1. Upload a CSV file containing stock price data
2. The application will calculate technical indicators and train a model
3. Make predictions using the trained model
4. Display forecast using Prophet
